# 📚 Library Management System

A simple Python-based Library Management System created for learning purposes.

---

## 🚀 Features

* Add Books
* Show Books
* Issue Books
* Return Books

---

## 📌 Functionality Added

* Books are stored using **dictionary**
* While issuing a book:

  * Student name is stored
  * Number of days is stored
* While returning:

  * If returned late → fine is applied
  * Fine system:

    * 1st week → ₹10/day
    * 2nd week → ₹20/day
    * 3rd week → ₹30/day

---

## 🛠️ Technologies Used

* Python

---

## ▶️ How to Run

1. Open terminal
2. Run the program:

```bash
python main.py
```

---

## 📂 Project Structure

* `main.py` → Menu system
* `add_books.py` → Add book
* `show_books.py` → Display books
* `issue_books.py` → Issue book
* `return_books.py` → Return book
* `utils.py` → Data storage
